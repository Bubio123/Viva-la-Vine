Datenschutzerklärung

Diese Datenschutzerklärung beschreibt, wie das Viva-la-Vine Plugin („das Plugin“) Daten verarbeitet.

1. Datenerhebung

Das Plugin erhebt oder speichert keine personenbezogenen Daten. Es werden lediglich für die Funktionalität des Plugins notwendige Informationen verwendet, die lokal auf Ihrem Gerät verarbeitet werden.

2. Weitergabe von Daten

Es erfolgt keine Weitergabe von Daten an Dritte. Da keine Daten erhoben werden, werden auch keine Daten mit anderen Parteien geteilt.

3. Sicherheit

Die Sicherheit Ihrer Daten ist wichtig. Da keine Daten gespeichert werden, gibt es keine Sicherheitsrisiken in Bezug auf Ihre personenbezogenen Informationen.

4. Änderungen an dieser Erklärung

Diese Datenschutzerklärung kann jederzeit ohne Vorankündigung geändert werden. Die aktuelle Version ist immer auf der offiziellen Webseite des Plugins verfügbar.
